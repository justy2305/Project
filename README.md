# Personal Portfolio Website

This is my personal portfolio website built to showcase who I am, my skills, and my learning journey as a beginner web developer.

## Technologies Used

- **HTML** – Structure and content of the website  
- **CSS** – Styling, layout, and theme design  
- **JavaScript** – Interactivity and form validation  

## Project Structure

The website consists of multiple separate files, which are:

- **Home Page** – Introduction  
- **About Page** – Information about me  
- **Technical Skills Page** – Technologies and skills I am learning  
- **Contact & More Information Page** – Contact form and additional details  

## Features

- **Dark Theme Mode** for a modern user experience  
- **Form Validation with Dynamic Feedback** using JavaScript  
- **Background Images** applied to:
  - Home page  
  - Contact details page  
- **Consistent Styling** across all pages using CSS  
- **code toggles the theme and remembers the user’s choice using localStorage** using JavaScript    

## Purpose

This project was created to practice and apply the fundamentals of **HTML, CSS, and JavaScript**, and to build a functional, responsive, and visually appealing portfolio website.

## Future Improvements

- Improve responsiveness for all screen sizes  
- Add more interactive JavaScript features  
- Enhance accessibility and performance
